# ES管理ノート 📋

就活のエントリーシート（ES）を企業ごとに保存・閲覧できるWebアプリです。

🔗 **アプリを使う**: GitHub Pages で公開後、`https://<ユーザー名>.github.io/<リポジトリ名>/` からアクセスできます。

## 機能

- 企業ごとに複数の設問・回答をセットで管理
- 文字数の自動カウントと字数制限チェック（OK / オーバー表示）
- 提出締切の管理（締切が近いと警告表示）
- ステータス管理（下書き / 提出済 / 通過 / 落選）
- 企業名・設問・回答の全文検索、ステータス絞り込み、並べ替え
- 回答のワンクリックコピー
- JSON形式でのバックアップ書き出し / 読み込み

## データの保存について

データはブラウザの `localStorage` に保存されます。サーバーには送信されません。
端末やブラウザを変えるとデータは引き継がれないため、「書き出し」機能でバックアップを取り、新しい環境で「読み込み」してください。

## 技術

HTML / CSS / JavaScript のみの単一ファイル構成（`index.html`）。ビルド不要。
