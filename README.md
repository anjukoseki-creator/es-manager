# 就活メモリー 🌸（旧：ES管理ノート）

就活のエントリーシート（ES）と選考状況を企業ごとにまとめて管理できるWebアプリです。

🔗 **アプリを使う**: https://anjukoseki-creator.github.io/es-manager/

## 機能

### ES管理
- 企業ごとに複数の設問・回答をセットで管理
- 文字数の自動カウントと字数制限チェック（OK / オーバー表示）
- 提出締切の管理（締切が近いと警告表示）
- 回答のワンクリックコピー
- 企業名・設問・回答の全文検索、絞り込み、並べ替え

### 貼り付け取り込み（AI解析）
- 応募フォームのテキストを丸ごとコピペすると、設問・回答・字数制限ごとに自動分解して登録
- ⚙️設定でGoogle Gemini APIキーを登録するとAI解析が有効（キーはブラウザ内にのみ保存）
- キーがなくてもパターン認識による自動解析が動作

### 選考管理
- 結果ステータス：下書き / 提出済 / 通過 / 落選
- 選考状況：ES提出 → 一次面接 → 二次面接 → 最終面接 ＋ 合格待ち
- 企業ホームページURLの登録・ワンクリックで開く

### ステータス別テーマ
- 通過 🌸 桜が舞う華やかなデザイン
- 落選 🥀 雨のしんみりしたデザイン
- 提出済 💓 鼓動するドキドキデザイン

### バックアップ
- JSON形式での書き出し / 読み込み

## データの保存について

データはブラウザの `localStorage` に保存されます。サーバーには送信されません。
端末やブラウザを変えるとデータは引き継がれないため、「書き出し」機能でバックアップを取り、新しい環境で「読み込み」してください。

## 技術

HTML / CSS / JavaScript のみの単一ファイル構成（`index.html`）。ビルド不要。
