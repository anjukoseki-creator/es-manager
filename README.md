# 就活メモリー 🌸

就活のエントリーシート（ES）と選考状況を企業ごとにまとめて管理できるWebアプリです。
AI自動整理・リアルタイム共有・自己分析メモまで、就活に必要な機能をひとつに。

🔗 **アプリを使う**: https://anjukoseki-creator.github.io/es-manager/

---

## 機能一覧

### 🤖 AI自動整理（貼り付け取り込み）
- 応募フォームのESテキストを丸ごとコピペすると、設問・回答・字数制限ごとに自動分解して登録
- Google Gemini APIキーを登録するとAI解析が有効（⚙️ 設定から登録）
- APIキーなしでもパターン認識による自動解析が動作

### 📋 ES管理
- 企業ごとに複数の設問・回答をセットで管理
- 設問ごとに**添削コメント**を記録・追跡（改善点をメモして書き直しを管理）
- 文字数の自動カウントと字数制限チェック（OK / オーバー表示）
- 提出締切の管理（締切が近いと警告表示）
- 回答のワンクリックコピー

### 🔗 リアルタイム共有
- 「共有」ボタンで短いURLを生成（約60文字）
- URLを送るだけで就活仲間・先輩・キャリアセンターとデータをリアルタイム同期
- 登録・ログイン不要。MQTT over WebSocketで実現

### 📊 選考管理
- **選考種別**：インターン / 本選考 を企業名の横で管理
- **結果ステータス**：下書き / 提出済 / 通過 / 落選
- **選考状況**：ES提出 → Webテスト → GD選考 → 一次面接 → 二次面接 → 最終面接 → 合格待ち
- 企業ホームページURLの登録・ワンクリックで開く
- カードを**長押しでドラッグ**して自由に並び替え

### 🎨 ステータス別テーマ
- 通過 🌸 桜が舞う華やかなデザイン
- 落選 🥀 雨のしんみりしたデザイン
- 提出済 💓 鼓動するドキドキデザイン

### 👤 自分の情報パネル
就活の軸となる自己分析をまとめて保存できるサイドパネル（右上「👤 自分の情報」から開く）
- ガクチカ / 強み / 弱み / 自己PR / 価値観
- 就活軸 / 将来像・キャリアビジョン / 志望業界・職種
- **エピソード辞典**：ES作成で使い回せるエピソードをタイトル付きで複数ストック

### 💾 バックアップ
- JSON形式での書き出し / 読み込み
- データはブラウザ内に自動保存。端末変更時にバックアップとして活用

---

## データの保存と安全性

| 機能 | データの扱い |
|------|------------|
| 通常使用 | ブラウザの `localStorage` に保存。外部送信なし |
| AI機能使用時 | ESテキストがGoogle（Gemini API）に送信される |
| 共有機能使用時 | HiveMQ公開ブローカーを経由。URLを知る人のみアクセス可 |

> 共有URLは信頼できる人にのみ送りましょう。

---

## 使い方

1. URLにアクセス（インストール・登録不要）
2. 「＋ 新しい企業」から企業を追加
3. 「📋 貼り付けて取り込み」でESを貼り付けてAI整理 or 手動で設問を追加
4. 各設問に回答・添削コメントを入力
5. 「👥 共有」で仲間とリアルタイム共有

---

## 技術

- HTML / CSS / JavaScript のみの単一ファイル構成（`index.html`）。ビルド不要
- AI解析：Google Gemini API
- リアルタイム共有：MQTT over WebSocket（HiveMQ公開ブローカー）
- データ保存：ブラウザの `localStorage`
