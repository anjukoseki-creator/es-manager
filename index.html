📖 このアプリを使う前に

✨ アプリの概要

「就活メモリー」は、ES（エントリーシート）の管理・添削・共有をまとめてできる無料のWebアプリです。

  • 🏢 企業ごとにES・選考状況を管理
  • 🤖 ESを貼り付けるだけでAIが設問ごとに自動整理
  • ✏️ 設問ごとに添削コメントを記録・追跡
  • 🔗 URLを送るだけで就活仲間とリアルタイム共有

📝 使い方

  1. 「＋ 新しい企業」ボタンから企業を登録
  2. 企業名・選考ステータス・締切日を入力
  3. 「📋 貼り付けて取り込み」でESをAIに整理させる（または手動で設問を追加）
  4. 各設問に回答・添削コメントを入力
  5. 「👥 共有」ボタンでURLを生成し、仲間に送る

🔒 仕組みと安全性

通常使用時：データはブラウザ（localStorage）内にのみ保存されます。外部サーバーには一切送信されません。

✅ 基本的に安全に使えます。データはあなたのブラウザ内にのみ保存され、外部には送信されません。

📌 AI機能を使う場合：ESのテキストがGoogleのサーバーで処理されます。氏名・住所・電話番号などの個人情報は含めないようにしましょう。ESの内容だけなら問題ありません。

📌 共有機能を使う場合：共有URLを知っている人なら誰でもデータを見られます。URLは信頼できる人だけに送りましょう。見知らぬ人や不特定多数への公開はやめてください。

🔑 Gemini APIキーの取得方法

AI機能（ES自動整理）の使用にはGoogleのGemini APIキーが必要です。

  1. Google AI Studio にアクセス（Googleアカウントでログイン）
  2. 「Get API Key」→「Create API key」でキーを発行
  3. アプリ右上の ⚙️ ボタンを押してAPIキーを入力・保存

※ APIキーはブラウザ内にのみ保存され、外部に漏洱することはありません。無料枠あり。

データはこのブラウザ内に自動保存されます。念のため定期的に「書き出し」でバックアップを取るのがおすすめです。
